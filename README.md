# ShadowNote · 随影笔记

> Notes that follow the video — 随视频流动的学习笔记。

一个**单文件、纯本地**的网页工具：把视频和**带时间戳的笔记**绑在一起，笔记随播放进度自动切换（像字幕，但显示的是你自己的学习导读）。还可以**用逐字稿一键生成笔记**，并对每一段内容**向 AI 追问**。

零安装、零后端、不上传任何数据——下载一个 HTML 双击就能用。

## 在线 Demo

<https://firmamento73.github.io/ShadowNote/>

（Demo 只是空播放器，视频和笔记需要你自己在本地载入，见下文。）

## 特性

- **视频同步笔记**：内置播放器，笔记随进度自动切换；目录可点击跳转；面板可拖动、缩放、最小化。
- **快进 / 快退控件**：底栏一键 ⏪ / ⏩，步长可选 5 / 10 / 15 / 30 / 60 秒；也支持键盘 `←` `→`（或 `J` `L`）跳转。
- **工作台生成笔记**：导入逐字稿（`.docx` / `.srt` / `.vtt` / `.txt`，或直接粘贴）→ 填主题 → 一键生成，或复制 Prompt 到任意模型手动跑。可选的逐字稿清理、可编辑可保存的 Prompt、生成进度条与停止。
- **「仅清理逐字稿」独立模式**：不生成笔记，只把逐字稿去口水、纠错、分段后单独下载；上次结果可随时再次下载。
- **多套 API 预设，按阶段分工**：可保存多套预设（各存 地址 + Key + 模型），随意命名、切换、删除。**清理、生成、问答可分别指定不同的服务商 / 模型**——例如用便宜模型清理、强模型生成、带视觉的模型做问答。
- **AI 追问（可选）**：载入 `notes.json` 后，对每段内容提问；自动概括 + 给出可继续追问的问题、平行对话线、上下文可在**本段 / 本段及以前 / 全文 / 不带原文**之间切换、数学公式用 KaTeX 渲染、问答可导出为 Markdown。
- **问答支持附件（多模态）**：追问时可 📎 添加图片 / 文本文件，或直接 `Ctrl+V` 粘贴截图；图片发给支持视觉的模型，文本文件作为上下文一并带入。
- **两种笔记形态**：`notes.txt`（纯笔记，轻量省 token）/ `notes.json`（含原文，开启追问模式）。
- **本地优先**：纯静态网页，没有后端；API key 只存在你自己的浏览器里。

## 快速开始

1. 下载 `随影笔记.html`，用现代浏览器（Chrome / Edge / Firefox）打开。
2. 点右上角「载入视频」选一个本地 mp4（或拖进窗口）。用 `←` `→`（或 `J` `L`）可快退 / 快进。
3. 点「载入笔记」选一个 `.txt` 或 `.json` 笔记文件。笔记会随视频自动同步。
4. 没有笔记？点右上角「工作台」，用逐字稿生成。

## 笔记格式

**`.txt`（纯笔记）** — 每段以时间戳开头，`#` 行为注释：

```
[00:00] 开场 · 这段在讲什么
重点：带着这个问题听……
逐字稿：这段的精简讲解……
```

**`.json`（开启追问模式）** — 每段多带 `raw`（原始逐字稿），供 AI 追问使用：

```json
{
  "version": 1,
  "segments": [
    { "t": "00:00", "title": "标题", "focus": "重点", "note": "讲解", "raw": "原始逐字稿" }
  ]
}
```

工作台里可下载这两种空白模板照着填。

## 开启 AI 追问（BYOK）

追问与一键生成都需要你**自带 API key**（Bring Your Own Key）：在「工作台 → 模型设置」里新建预设、选服务商、填 Base URL 与 key（兼容 OpenAI 接口），保存后即可在清理 / 生成 / 问答各处选用。可存多套预设自由切换。

> ⚠ 浏览器直连云端 API 是否成功取决于各家的 CORS 策略。若被拦截，可改用本地模型（如 Ollama `http://localhost:11434/v1`）、用本地服务器打开本页，或走「复制 Prompt」手动方式。

## 隐私

- 没有任何后端，所有处理都在你的浏览器里完成。
- 视频、笔记、逐字稿都**不会上传**。
- API key 仅保存在浏览器 `localStorage`，请求直接从你的浏览器发往模型服务商，作者和任何中间服务器都接触不到。

## 依赖

**公式引擎 KaTeX 已内置到文件中**（含 JS / CSS / 字体，base64 打包），因此**公式渲染完全离线可用**，无需联网。

仍通过 CDN 加载的仅剩：

- [mammoth.js](https://github.com/mwilliamson/mammoth.js)（解析 `.docx`，BSD-2-Clause）
- Google Fonts：Newsreader、IBM Plex Mono（纯装饰性字体，开放字体许可）

离线时：公式正常渲染，仅**无法解析 `.docx`**（可改用 `.srt` / `.vtt` / `.txt` 或直接粘贴），字体回退到系统字体。核心的播放、笔记同步、生成与问答均不受影响。
