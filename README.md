# ShadowNote · 随影笔记

> Notes that follow the video — 随视频流动的学习笔记。

一个**单文件、纯本地**的网页工具：把视频和**带时间戳的笔记**绑在一起，笔记随播放进度自动切换（像字幕，但显示的是你自己的学习导读）。还可以**用逐字稿一键生成笔记**，并对每一段内容**向 AI 追问**。

零安装、零后端、不上传任何数据——下载一个 HTML 双击就能用

## 在线 Demo

`https://firmamento73.github.io/shadownote/`

（Demo 只是空播放器，视频和笔记需要你自己在本地载入，见下文。）

## 特性

- **视频同步笔记**：内置播放器，笔记随进度自动切换；目录可点击跳转；面板可拖动、缩放、最小化。
- **工作台生成笔记**：导入逐字稿（`.docx` / `.srt` / `.vtt` / `.txt`，或直接粘贴）→ 填主题 → 一键生成，或复制 Prompt 到任意模型手动跑。可选的逐字稿清理、可编辑可保存的 Prompt、生成进度条与停止。
- **AI 追问（可选）**：载入 `notes.json` 后，对每段内容提问；自动概括 + 给出可继续追问的问题、平行对话线、按段/全文/选段切换上下文、数学公式用 KaTeX 渲染、问答可导出为 Markdown。
- **两种笔记形态**：`notes.txt`（纯笔记，轻量省 token）/ `notes.json`（含原文，开启追问模式）。
- **本地优先**：纯静态网页，没有后端；API key 只存在你自己的浏览器里。

## 快速开始

1. 下载 `随影笔记.html`（或 `index.html`），用现代浏览器（Chrome / Edge / Firefox）打开。
2. 点右上角「载入视频」选一个本地 mp4（或拖进窗口）。
3. 点「载入笔记」选一个 `.txt` 或 `.json` 笔记文件。笔记会随视频自动同步。
4. 没有笔记？点右上角「工作台」，用逐字稿生成。

## 笔记格式

**`.txt`（纯笔记）** — 每段以时间戳开头，`#` 行为注释：

```
[00:00] 开场 · 这段在讲什么
重点：带着这个问题听……
逐字稿：这段的精简讲解……
```

**`.json`（开启追问模式）** — 每段多带 `raw`（原始逐字稿），供 AI 追问使用：

```json
{
  "version": 1,
  "segments": [
    { "t": "00:00", "title": "标题", "focus": "重点", "note": "讲解", "raw": "原始逐字稿" }
  ]
}
```

工作台里可下载这两种空白模板照着填。

## 开启 AI 追问（BYOK）

追问与一键生成都需要你**自带 API key**（Bring Your Own Key）：在「工作台 → 模型设置」选服务商、填 Base URL 与 key（兼容 OpenAI 接口）。

> ⚠ 浏览器直连云端 API 是否成功取决于各家的 CORS 策略。若被拦截，可改用本地模型（如 Ollama `http://localhost:11434/v1`）、用本地服务器打开本页，或走「复制 Prompt」手动方式。

## 隐私

- 没有任何后端，所有处理都在你的浏览器里完成。
- 视频、笔记、逐字稿都**不会上传**。
- API key 仅保存在浏览器 `localStorage`，请求直接从你的浏览器发往模型服务商，作者和任何中间服务器都接触不到。

## 依赖

通过 CDN 加载，均为宽松许可：

- [KaTeX](https://katex.org/)（公式渲染，MIT）
- [mammoth.js](https://github.com/mwilliamson/mammoth.js)（解析 .docx，BSD-2-Clause）
- Google Fonts：Newsreader、IBM Plex Mono（开放字体许可）

离线时这些会优雅降级（公式不渲染、无法解析 docx、字体回退），核心的播放与笔记同步仍可用。
